At the core of this role, a security engineer keeps an organization's security systems up and running. Let's learn more about what that means. Security engineers are a technical role in an organization. Security by design is an approach that makes sure every step of your IT or software lifecycle process has security in mind. Think of it like building a house from the ground up. You want to make sure your house is strong, can withstand storms, and has measures in place against intruders. Security by design takes the same approach and is the blueprint in making sure every system you set up and every component you incorporate has been looked at through a security lens. The implementation aspect of [[Security Engineering]] builds upon this blueprint and typically involve using a combination of policies, processes, and technologies to enforce your security requirements. Examples of an implementation can look like installing and configuring firewalls to monitor and filter your network traffic.
