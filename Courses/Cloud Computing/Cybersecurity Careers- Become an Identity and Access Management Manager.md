You might be asked to tell me when you were assigned a task that you were unfamiliar with, what did you do, and what was the result. You could say that once you took over a task where the goal was to let users log into a server using a smart card, the job was something I had never done before and the first thing I did was understand the encryption used in the smart card. I then looked at the server's user guide to confirm the encryption protocol is accepted. I then used trial and error to test out the different scenarios, and after two weeks of testing, I was able to show leadership a demo of showing a user using their smart card to log into the server. The situation was letting users log into a server using their smart card versus only using a username and password. The task was figuring out how the server could accept a user's smart card
