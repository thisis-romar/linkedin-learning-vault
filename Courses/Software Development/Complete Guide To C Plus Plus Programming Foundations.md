C++ uses the [[Object-Oriented Programming (OOP)|object-oriented programming]] paradigm, just like some modern programming languages like [[Java]] and [[Python (Programming Language)|Python]]. C++ was created by computer scientist Bjarne Stroustrup in 1979, and it was released in 1985. But please keep in mind that C++ is not an extension of C, so there are some important differences between the two languages.
