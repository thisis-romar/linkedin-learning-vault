>
> **[3:06](https://www.linkedin.com/learning/cybersecurity-with-cloud-computing-2/logging-and-monitoring-in-the-cloud?u=76281980&t=186)** for customers to run. But in that paper SOC as a Service for cloud computing environments, Alruwaili and Gulliver go further and present a model of a full Security Operations Center in the cloud. This model provides for log sources from the cloud services, as well as scanning and physical security monitoring. The proposed Security Operations Center as a Service model is designed to provide the comprehensive information security in event management for all cloud devices and applications. The author's proposed the model be run by a trusted third-party, encouraging cloud providers to invest in service functionality and security. The model doesn't however, consider on-premise integration issues. Security and [Telecommunications](../../Skills/Network%20and%20System%20Administration/Telecommunications.md) vendors are beginning to promote SOC as a Service as seen in the report on the service, launched by Macquarie Telecommunications, in September 2019.
