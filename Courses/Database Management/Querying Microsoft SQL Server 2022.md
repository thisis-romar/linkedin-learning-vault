[[SQL]], often pronounced as sequel, stands for Structured Query Language. But not all versions of the SQL language are exactly the same. As companies added differentiating features to their database product to make them more enticing from those of their competitors, each company introduced unique features and capabilities to the RDBMS platform that they brought to market. A specialized function here, a new command there, and eventually the common SQL syntax diverged into different implementations.
