The first summary metric is sensitivity, which is the ratio of how many values were correctly identified as positive that were actually positive. Next is specificity, which is the ratio of how many values were correctly identified as negative that were actually negative. It has values from zero to one with zero being no correctly predicted values, and one being all correctly predicted values. Next is f-measure, which is a weighted harmonic mean of recall and precision. It is made with the formula two multiplied by precision, multiplied by recall, divided by precision, plus recall. It has values from zero to one with zero being no correctly predicted values, and one being all correctly predicted values. Finally, there's Cohen's kappa, which determines how much better a model is versus a random classifier, such as random chance. This coefficient ranges from negative one to one with values negative one to zero meaning no agreement, and one meaning perfect agreement.
