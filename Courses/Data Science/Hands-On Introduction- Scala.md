Of course, Scala is a programming language. It was first released in 2004, created by Martin Odersky who is an academic at a Swiss university. Scala is a combination of the words scalable and language because it was designed to grow with the demands of users. Scala compiles down to [Java](../../Skills/Software%20Development/Java.md) bytecode and runs on the Java virtual machine. This means that we can run Java alongside Scala in the same project. One of the reasons behind Scala's success is its take up in the Java community as Scala was designed in part to improve some of the weaknesses of Java. Scala is a lot terser than Java and we'll discover other benefits shortly. So what differentiates Scala from other languages? First, Scala combines both object-oriented and [Functional Programming](../../Skills/Software%20Development/Functional%20Programming.md) styles in one language. You might have come across the object-oriented paradigm where everything is an object in languages like [Python](../../Skills/Software%20Development/Python%20(Programming%20Language).md) and [JavaScript](../../Skills/Software%20Development/JavaScript.md). Functional programming aims to eliminate mutability. We can reassign the values of mutable variables. However, functional programming does not permit this. When a value is assigned to an immutable variable, we can't reassign it. We have to create a new one instead.
>
> **[1:33](https://www.linkedin.com/learning/hands-on-introduction-scala/scala-introduction?u=76281980&t=93)** This makes our code less prone to bugs and easier to test. Second, Scala is a statically typed language. This means that variables are assigned types. These types are checked at compile time, and this can help us to catch bugs. The compiler can often infer types, so we don't need to annotate all of our code with them. Don't worry if this doesn't make sense yet. We'll write Hello World in Scala in one of my next videos and you'll understand exactly what I mean. Third, Scala is changing. The latest version of Scala is Scala 3, which was released in 2021, and this course we'll use the latest version of Scala, Scala 3.
