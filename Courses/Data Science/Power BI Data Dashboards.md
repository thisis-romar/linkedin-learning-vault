We then add values of red, green, and blue on top of the black to get a color value that's visible to the human eye. For the most part, the CMYK and RGB color models produce similar colors, but we use them in different mediums. For our Power BI data visualizations, we're going to refer to color values from the RGB color model. More specifically, we're going to use hexadecimal representations of these color values. You might already be familiar with these color values. They start with a hash or pound sign, followed by a string of six letters or numbers representing their color value.
