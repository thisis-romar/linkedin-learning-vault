## How to Access

### Browse on GitHub (clickable links)
Switch to the **[github-browse](../../tree/github-browse)** branch — all `[[wikilinks]]` are auto-converted to standard markdown links so you can click through notes directly on GitHub.

### Browse on the web (full experience)
Visit **[thisis-romar.github.io/linkedin-learning-vault](https://thisis-romar.github.io/linkedin-learning-vault)** — a Quartz-powered site with graph view, backlinks, full-text search, and wikilink navigation.

### Open in Obsidian (recommended)
```bash
git clone https://github.com/thisis-romar/linkedin-learning-vault.git
```
1. Open Obsidian → "Open folder as vault" → select the cloned folder
2. Trust the vault when prompted
3. Verify CSS snippets are enabled: Settings → Appearance → CSS snippets → toggle `timestamps` and `note-types`
4. Press `Ctrl+G` / `Cmd+G` to open the graph view (20 color groups pre-configured)

## Branch Structure

| Branch | Purpose |
